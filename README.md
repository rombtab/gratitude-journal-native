
# Gratitude Journal (Native)
Мобильное приложение для ведения дневника благодарности, созданное с помощью React Native и Expo.

## Цели проекта
*   Освоить полный цикл разработки под Android.
*   Создать работающее и опубликованное приложение.
*   Изучить современную архитектуру React Native-проектов.

## Стек технологий
*   React Native
*   Expo
*   TypeScript
*   Zustand (для управления состоянием)

## Установка и запуск
1.  Клонируйте репозиторий: `git clone https://github.com/rombtab/gratitude-journal-native.git`
2.  Установите зависимости: `npm install`
3.  Запустите проект: `npx expo start`
